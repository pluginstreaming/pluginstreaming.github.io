# PluginStreaming Kodi Repository

Este repositório publica uma fonte Kodi em `https://pluginstreaming.github.io` para instalar e atualizar automaticamente os addons FILMOM, PLUGINSTREAMING e PLUGINSTREAMING VIP.

## URL para o Gestor de Arquivos do Kodi

```text
https://pluginstreaming.github.io
```

## Pacote inicial do repositório

```text
https://pluginstreaming.github.io/zips/repository.plugnstreaming.vip-1.0.0.zip
```
